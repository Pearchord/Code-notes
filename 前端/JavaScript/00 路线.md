## 📝 补充：`.flatMap()` 快速说明

教程里没有这个，但很简单——它就是 **先 `.map()` 再 `.flat()`**：

````javascript
// map 返回嵌套数组
const result = [[1, 2], [3, 4]].map(arr => arr)
// → [[1, 2], [3, 4]]

// flatMap 会自动"拍平"一层
const result2 = [[1, 2], [3, 4]].flatMap(arr => arr)
// → [1, 2, 3, 4]

// 实际用途：一个用户有多个标签，提取所有标签
const users = [
  { name: '小明', tags: ['前端', 'Vue'] },
  { name: '小红', tags: ['后端', 'Go'] },
]

const allTags = users.flatMap(user => user.tags)
// → ['前端', 'Vue', '后端', 'Go']
````
