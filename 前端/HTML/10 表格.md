
给列上色
```
<table>
  <colgroup>
    <col />
    <col style="background-color: yellow" />
  </colgroup>
  ……

```
一个col代表一列，示例为给第二列上色
同样可以span

添加 <thead>、<tbody> 和 <tfoot> 结构
可以实现更多功能，比如用CSS实现在页面滑动时始终显示表头。

```
  <tr>
    <td id="nested">
      <table id="table2">
        <tr>
          <td>单元格 1</td>
          <td>单元格 2</td>
          <td>单元格 3</td>
        </tr>
      </table>
    </td>
```
