## 任务 3：一条聊天消息 ⭐⭐

**要求：**
- 模仿微信的聊天气泡
- 做两条消息：一条靠左（别人发的），一条靠右（自己发的）
- 颜色不同

**参考答案：**

````html
<div class="min-h-screen bg-gray-100 p-6">
  <div class="max-w-md mx-auto space-y-4">

    <!-- 别人发的消息（靠左） -->
    <div class="flex items-start gap-3">
      <div class="w-10 h-10 bg-gray-400 rounded-full flex-shrink-0"></div>
      <div class="bg-white rounded-2xl rounded-tl-none px-4 py-2 shadow-sm">
        <p class="text-sm text-gray-800">你好啊，今天怎么样？</p>
      </div>
    </div>

    <!-- 自己发的消息（靠右） -->
    <div class="flex items-start gap-3 flex-row-reverse">
      <div class="w-10 h-10 bg-blue-500 rounded-full flex-shrink-0"></div>
      <div class="bg-blue-500 rounded-2xl rounded-tr-none px-4 py-2 shadow-sm">
        <p class="text-sm text-white">挺好的！在学 Tailwind CSS 😄</p>
      </div>
    </div>

  </div>
</div>
````

**学到的知识点：**

| class | 含义 |
|-------|------|
| `max-w-md mx-auto` | 最大宽度 + 水平居中 |
| `space-y-4` | 子元素之间垂直间距 16px |
| `flex-row-reverse` | 水平反向排列（让消息靠右） |
| `rounded-2xl rounded-tl-none` | 圆角但左上角不圆（气泡效果） |
| `items-start` | 顶部对齐 |

## 任务 4：聊天输入框 ⭐⭐⭐

**要求：**
- 底部固定一个输入区域
- 左边一个输入框，右边一个发送按钮
- 按钮有 hover 效果

**参考答案：**

````html
<div class="min-h-screen bg-gray-100 flex flex-col">
  <!-- 消息区域 -->
  <div class="flex-1 p-4">
    <p class="text-center text-gray-400 mt-20">暂无消息</p>
  </div>

  <!-- 底部输入栏 -->
  <div class="bg-white border-t border-gray-200 p-3 flex items-center gap-3">
    <input
      type="text"
      placeholder="输入消息..."
      class="flex-1 border border-gray-300 rounded-full px-4 py-2 text-sm
             focus:outline-none focus:border-blue-500 focus:ring-2 focus:ring-blue-200"
    />
    <button
      class="bg-blue-500 text-white px-5 py-2 rounded-full text-sm font-medium
             hover:bg-blue-600 active:bg-blue-700 transition-colors"
    >
      发送
    </button>
  </div>
</div>
````

**学到的知识点：**

| class | 含义 |
|-------|------|
| `flex flex-col` | 纵向弹性布局 |
| `flex-1` | 占据剩余空间 |
| `border-t` | 只有顶部边框 |
| `focus:outline-none` | 聚焦时去掉默认边框 |
| `focus:ring-2 focus:ring-blue-200` | 聚焦时显示蓝色光环 |
| `hover:bg-blue-600` | 鼠标悬停变深 |
| `active:bg-blue-700` | 点击时更深 |
| `transition-colors` | 颜色变化有动画过渡 |
