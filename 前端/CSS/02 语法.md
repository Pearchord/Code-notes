改变元素的默认行为
```
li {
  list-style-type: none;
}
```
不仅能移除项目符号，而 `square`值还能把默认的小黑球变成方框框。

使用类名：
给 HTML 元素加个类名（class），再选中那个类名
```
<ul>
  <li>项目一</li>
  <li class="special">项目二</li>
  <li>项目 <em>三</em></li>
</ul>
```
要选中这个 `special` 类，只需在选择器的开头加个西文句点（.）
```
.special {
  color: orange;
  font-weight: bold;
}
```

有时你会发现选择器中，HTML 元素选择器跟类一起出现：
```
li.special {
  color: orange;
  font-weight: bold;
}
```

这是选中每个 special 类的 li 元素。它对 span 还有其他元素不起作用了。

根据元素在文档中的位置确定样式：
```
li em {
  color: rebeccapurple;
}
```
`<li>`内部的`<em>`元素变为紫色

**相邻选择符**：
选中标题后且与标题具有相同层级的段落样式


根据状态确定样式：
```
a:link {
  color: pink;
}

a:visited {
  color: green;
}
```

鼠标悬停的时候移除下划线
```
a:hover {
  text-decoration: none;
}
```
