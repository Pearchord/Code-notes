存否和值选择器（标签属性选择器、存在性选择器）
```
<ul>
  <li>Item 1</li>
  <li class="a">Item 2</li>
  <li class="a b">Item 3</li>
  <li class="ab">Item 4</li>
</ul>




li[class] {
  font-size: 120%;
}

li[class="a"] {
  background-color: yellow;
}

li[class~="a"] {
  color: red;
}
```
[class~="a"]会匹配包含a类的值，它选中了第二和第三项。

tips:
```
class="a b"（多个标签）
这表示该元素同时拥有两个独立的类名：“a”和“b”。

核心逻辑：空格在 class 属性里是“分隔符”。

CSS 匹配：它会同时继承 .a 和 .b 定义的所有样式。如果两个类里有冲突的属性（比如都改了背景色），则取决于 CSS 文件中谁写在后面。

用途：这是 CSS 中最常用的组合模式。例如，“a”负责控制大小，“b”负责控制颜色。
```

还有一种特定前缀选择器：如div[lang|="zh"]的会选中 `lang="zh"` 和 `lang="zh-CN"`
只认“完全一致”或者“前缀加横杠”


子字符串匹配选择器：
li[class^="a"]匹配了任何值开头为a的属性。
li[class$="a"]匹配了任何值结尾为a的属性。
li[class*="a"]匹配了任何值的字符串中出现了a的属性。

大小写不敏感：
后面加个i
```
li[class^="a" i] {
  color: red;
}
```
