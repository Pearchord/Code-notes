选择器列表，加“，”
```
h1, 
.special {
  color: blue;
}
```
当使用选择器列表时，如果任何一个选择器无效 (存在语法错误)，那么整条规则都会被忽略。
如：
```
h1,
..special {
  color: blue;
}
```

类型、类和 ID 选择器：
```
h1 {
}

.box {
}

#unique {
}
```
标签属性选择器（存在性选择器）：
[[04 属性选择器]]
```
a[title] {
}

a[href="https://example.com"] {
}
```

a[href="https://google.com"]	必须是这个特定的 URL	最高（只选一个人）
a[title]	只要有 “提示信息” 就算	中等（选出一类人）
a	只要是超链接就算	最低（全选）

伪类与伪元素：
```
a:hover {
}

p::first-line {
}
```

示例的伪类为悬浮状态
示例的伪元素为p中的第一行

运算符：
[[06 关系选择器]]

```
article > p {
}
```

直接被 **`article`** 包裹的 **`p`** (“亲儿子”)
